#+TITLE: dotfiles

Hello! These are my personal dotfiles that manage my laptop's linux environment.

| Problem          | Solution           |
|------------------+--------------------|
| Operating System | NixOS 20.09        |
| Window Manager   | =exwm=             |
| Input Method     | =emacs + exwm-xim= |
| Editor           | =emacs=            |
| Notifications    | =dunst=            |
| Shell            | =fish=             |
| Color Scheme     | =pywal + wpgtk=    |
| Email            | =isync + mu4e=     |
| Web Browser      | =firefox=          |
| Networking       | =iwd=              |
| System Tray      | =stalonetray=      |
